# Mkdocs-exemplos

Este repositório serve como um exemplo prático das funcionalidades oferecidas pelo tema mkdocs-material,
apresentando uma abordagem direta e simplificada.

Além disso, pode ser utilizado como um modelo (template) para iniciar projetos que fazem uso do mkdocs-material.

## Requisitos

Para utilizar este projeto, certifique-se de ter os seguintes requisitos instalados:

- Docker
- Docker Compose

## Como usar

Para executar o projeto, basta estar na raiz do mesmo e executar o seguinte comando no terminal:

```sh
docker-compose up

# ou

docker compose up -d
```

Este comando iniciará o ambiente e tornará a documentação disponível localmente.
Após a execução bem-sucedida, você pode acessar a documentação no seu navegador,
normalmente através de [http://localhost:8000](http://localhost:8000).

Sinta-se à vontade para personalizar o projeto de acordo com suas necessidades.
